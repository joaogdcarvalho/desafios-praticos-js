# ROCKETSEAT STARTER - JAVASCRIPT

## Descrição do Projeto

Este repositório é um registro do curso introdutório de JavaScript oferecido pela Rocketseat, com foco na revisão de conceitos fundamentais da linguagem. O curso abrange desde noções básicas, como variáveis e operações matemáticas, até tópicos mais avançados, como manipulação da DOM e utilização de Promises e Axios para requisições assíncronas.

## Conteúdo do Projeto

### Aula 1

- **Desafios:**
  - Exercícios práticos (desafio01e01 a desafio01e05).
- **Projeto:**
  - Estruturação de arquivos e página inicial (index.html).

### Aula 2

- **Desafios:**
  - Exercícios práticos (desafio02e01 a desafio02e04).
- **Projeto:**
  - Desenvolvimento de funcionalidades específicas (index.html).

### Aula 3

- **Projeto:**
  - Implementação de funcionalidades utilizando JavaScript (todos.js).
  - Estruturação da página inicial (index.html).

### Aula 4

- **Desafios:**
